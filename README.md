# Dropbox Ignore

> Ignore Dropbox files (macOS only)

## Installation

```sh
git clone git@github.com:noahsug/dropbox-ignore.git
cd dropbox-ignore
yarn install
```

## Usage

Preview changes

```sh
yarn test
```

Tell Dropbox to ignore node_module files

```sh
yarn start
```

Note: this assumes your Dropbox path is `~/Dropbox`
